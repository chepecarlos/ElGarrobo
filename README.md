# ElGarrobo

Heramienta para aumentar la eficiencia y rapidez, en desarrollo y trabajo en produccion con diferentes heramientas, compatible con StreamDeck en Linux y multiples teclados

### Compatible con

-   Teclado USB
-   El Gato StreamDeck
-   OBS_WebSoket
-   control MQTT
-   Linux y Python3
